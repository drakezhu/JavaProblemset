# Java后端面试准备微信小程序

一个基于微信小程序开发的Java后端面试准备工具，与Web版本功能一致，帮助你随时随地复习Java面试题。

## 🚀 功能特性

- **按类别筛选** - 支持按Java核心、Spring、数据库等类别筛选问题
- **按难度筛选** - 支持按Easy、Medium、Hard难度级别筛选
- **分页浏览** - 避免无限滚动，每页显示固定数量的问题
- **详细查看** - 点击问题卡片查看完整答案和解析
- **响应式设计** - 适配不同尺寸的手机屏幕

## 📝 如何使用

### 前提条件
- 微信开发者工具
- 小程序AppID（可选，使用测试号也可）

### 开发步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd InterviewPrep/wechat-miniprogram
   ```

2. **在微信开发者工具中导入项目**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 填写项目名称和项目目录（指向 `wechat-miniprogram` 文件夹）
   - 选择AppID（使用测试号或自有AppID）
   - 点击「导入」

3. **配置数据文件**
   - 将 `questions.json` 文件复制到小程序根目录
   - 或修改 `app.js` 中的数据加载逻辑，使用远程API

4. **运行项目**
   - 在微信开发者工具中点击「编译」按钮
   - 可以在模拟器中查看效果
   - 也可以在真机上预览

## 🎨 页面说明

### 首页
- **筛选区域** - 显示所有类别和难度标签，点击可切换筛选条件
- **问题列表** - 以卡片形式显示问题，包含类别、难度和问题标题
- **分页控件** - 底部显示分页信息和页码切换按钮
- **每页显示设置** - 可选择每页显示6、9或12个问题

### 详情页
- **问题详情** - 显示完整的问题内容
- **答案内容** - 显示详细的答案解析
- **返回按钮** - 点击返回问题列表页

## 📄 数据结构

面试题数据结构如下：

```json
{
  "id": "唯一标识符",
  "question": "问题内容",
  "answer": "答案内容",
  "category": "类别名称",
  "difficulty": 1  // 1=Easy, 2=Medium, 3=Hard
}
```
